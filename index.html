Data Hall—▸
Each hall profile is one physical facility: site, elevation, air volume, installed plant (capabilities + rates), plus its real-world efficiency factor and current capacity derates. Switch tabs to move between buildings / halls. Import merges by hall name (same name replaces, new names append), so colleagues' files add to yours. The customer SLAs below are contracts evaluated inside the active hall at its barometric pressure.
Customer SLA—▸
SLA boundary = the binding constraint at each temperature: RH-max curve vs. dew-point cap (whichever is lower) on top, RH-min vs. dew-point floor on the bottom. Leave dew-point cap blank for an RH-only box. All boundaries recompute at site pressure. ASHRAE TC 9.9 reference ramp limits: 9°F/hr (5°C/hr) where tape media is present; 36°F/hr (20°C/hr) otherwise.
